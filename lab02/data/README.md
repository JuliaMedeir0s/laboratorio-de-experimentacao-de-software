# Dados (Lab02)

Este diretório contém arquivos gerados pelos scripts do `lab02/scripts/`.

- `repos.csv`: lista dos 1.000 repositórios Java mais populares (Sprint 1).
- `raw/`: saídas brutas do CK para múltiplos repositórios (geralmente não versionado).
- `processed/`: dados agregados/estatísticas (geralmente não versionado).
- `sample/`: amostra de saída do CK para 1 repositório (Sprint 1).
